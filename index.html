Project Overview

The purpose of this project is to designate a spot for one or more goods or things on the appropriate aisle and shelf in a warehouse. To give one or more objects the appropriate position and quantity, utilise the Assign position function. The precise place is found on a shelf in a warehouse aisle. The item picker visits the item location to pick it up before sending it out for delivery. To move one or more things with the necessary amounts from one area to another, utilise the Transfer area feature.It is possible to move any aisle, shelf, and location from one warehouse to another. In a similar manner, it is possible to move any or all of the Shelves and Locations from one aisle to another.

Item pickers can locate an item and identify the appropriate aisle, shelf, or location on a warehouse by using item pick lists. Different kinds of pick lists are made according to specific standards. The object might be marked as Picked, Missing, or Damaged by the item picker. Solution Designing

Assign Location

→

Assign Location is used to assign a location to one or more items with the desired quantities. The exact location is present on a Shelf in an Aisle of a Warehouse. Before sending an item out for the delivery, the Item Picker goes to the location of the items to collect it.

Transfer Location

→

Transfer Location is used to transfer one or more items with the desired quantities from 1 location to another. Multiple items can also be transferred to any location. All the Aisle along with the Shelves and Locations can be transferred from one Warehouse to another. Similarly, all the Shelves along with the Locations can be transferred from one Aisle to another. Lastly, the inventory allocation of the items can be changed/transferred by transferring the Location from one to another.

Item Picklists

→

Item Picklists are useful for Item Pickers to track down the item and find the corresponding Aisle/Shelf/Location on which it is placed in a Warehouse. Different types of Picklists are created based on certain criterias. There is a functionality to mark the item as Picked/Missing/Damaged by the Item Picker. For the products having the Missing/Damaged status a new Pack Slip is printed to validate the situation.
